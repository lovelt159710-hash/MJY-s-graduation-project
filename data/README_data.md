# Data instructions (no raw data uploaded)

This repo does NOT include raw data.

Place raw Excel files under your own `base_path` with the following folder names:
- Annual Table of Basic Information of Listed Companies/
- Monthly stock return rate file/
- Research and Development Investment Status Table/
- Balance Sheet/
- Income Statement/
- Three-factor model indicators (monthly)/

Then run:
source("paper_style_annotated_pipeline.R")
